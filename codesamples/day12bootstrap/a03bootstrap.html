Container (Fixed-width)

This is a fixed-width container. It adjusts its width based on the screen size but has predefined maximum widths at different breakpoints. For example, on large screens, it may be 1140px wide.


Container Fluid (Full-width)

This is a fluid container. It takes up the full width of the viewport, meaning it stretches from edge to edge of the screen at any screen size.


Container with 'sm' Breakpoint

This container will have a max-width that applies to the 'sm' (small) breakpoint and up. On smaller screens, it takes up 100% width, but on screens larger than 576px, it becomes constrained to a smaller max-width.


Container with 'md' Breakpoint

This container has a max-width that applies from the 'md' (medium) breakpoint and up. It behaves like a fixed-width container at medium and larger screen sizes.


Container with 'lg' Breakpoint

This container will have a maximum width at the 'lg' (large) breakpoint and above. It will adjust based on the screen size, but will remain constrained on larger screens.


Container with 'xl' Breakpoint

This container will have a max-width that applies at the 'xl' (extra-large) breakpoint. On smaller screens, it behaves like a full-width container, but becomes constrained as the screen size increases.
